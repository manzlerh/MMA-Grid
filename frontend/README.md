## Frontend

Planned **React + Vite** single-page app that will:

- Render the daily **UFC Grid** and **UFC Connections** games.
- Provide fighter search/autocomplete and interactive validation feedback.
- Display user stats such as streaks, completion status, and past scores.
- Use Tailwind CSS (or similar utility-first styling) for a UFC-inspired dark theme.

This folder will be scaffolded as a Vite/React app in a later step.
